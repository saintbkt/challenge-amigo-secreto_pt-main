Amigo Secreto
Este é um projeto simples de Amigo Secreto desenvolvido com JavaScript, HTML e CSS. A ideia é permitir que você adicione amigos à lista, atualize essa lista e faça o sorteio do amigo secreto aleatoriamente.

Funcionalidades
Adicionar Amigos: Você pode adicionar amigos à lista de sorteio digitando o nome e clicando em "Adicionar".
Visualizar Lista: A lista de amigos é atualizada automaticamente conforme novos nomes são adicionados.
Sorteio Aleatório: Você pode sortear um amigo secreto de maneira aleatória entre os amigos cadastrados.
Como Usar
Abra o arquivo index.html em seu navegador.
Na caixa de entrada, digite o nome de um amigo e clique em "Adicionar" para incluir o nome na lista.
Para sortear um amigo secreto, clique no botão "Sortear". O nome de quem foi sorteado será exibido.
Estrutura do Código
JavaScript: Responsável pelas funcionalidades de adicionar amigos à lista, atualizar a interface e realizar o sorteio.
HTML: Estrutura básica da página, incluindo os campos de entrada e os botões.
CSS (se houver): Estiliza a página para uma apresentação visual mais agradável.
Funções Implementadas
adicionarAmigo():
Adiciona um amigo à lista após validar que o campo não está vazio.
atualizarLista():
Atualiza a lista exibida na tela com os nomes dos amigos cadastrados.
sortearAmigo():
Realiza o sorteio aleatório de um amigo da lista e exibe o nome do sorteado.
